# Data-governance

Repository này chứa các tài liệu kiến trúc ở mức solution/platform cho các bài toán dữ liệu và AI quy mô lớn.

## Danh mục tài liệu

### 01. IoMT Platform Technical Architecture
- **File:** `01_iomt_platform_technical_architecture.md`
- **Vai trò:** tài liệu kiến trúc chính cho nền tảng IoMT phục vụ khám sức khỏe ở quy mô lớn
- **Mức độ:** solution architecture / platform architecture

### 02. Harness Engineering for Production AI Systems
- **File:** `02_harness_engineering.md`
- **Vai trò:** tài liệu bổ trợ cho lớp orchestration, tool execution, memory, approval và observability của hệ thống AI production
- **Mức độ:** application/platform architecture

## Mục tiêu của repository

Bộ tài liệu này tập trung vào các nội dung:

- kiến trúc kỹ thuật ở quy mô production,
- non-functional requirements,
- logical / runtime / deployment views,
- dữ liệu, bảo mật, khả năng mở rộng và vận hành,
- quyết định kiến trúc và trade-off.

Mục tiêu là tạo ra tài liệu đủ chiều sâu để dùng trong các bối cảnh như:

- kiến trúc giải pháp,
- review kỹ thuật,
- pre-sale kỹ thuật,
- planning cho implementation quy mô lớn,
- trao đổi giữa architect, tech lead và delivery team.
